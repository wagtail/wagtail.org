# Changelog

## [2.3.0] - 2026-02-10

### Changed

-   Upgraded Wagtail from 7.2.1 to 7.3.0
-   Updated package version to 2.3.0

### Upgrade Notes

-   Wagtail 7.3 introduces initial support for autosave functionality
-   New StructBlock form layout options are now available
-   Image quality defaults have been adjusted for better performance
-   Settings editing hooks are now available for customization

No code changes were required for this upgrade as the project does not use the affected features.
